# IBM HR Attrition Analysis

## Project Overview
This project analyzes employee attrition using Python, Machine Learning, and Power BI.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-Learn
- Power BI

## Project Steps
1. Data Cleaning
2. Exploratory Data Analysis
3. Feature Engineering
4. Machine Learning (Random Forest)
5. Interactive Power BI Dashboards

## Results
- Random Forest Accuracy: 88%
- Identified key factors affecting employee attrition
- Built interactive dashboards for HR decision-making
